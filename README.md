# Categories - E-Commerce App

A Flutter e-commerce application showcasing electronics and tech products by categories.

## Features

- **Category Browse**: Browse products across different categories (Phones, Laptops, Tablets, Watches, Headphones, Cameras, TVs, Gaming)
- **Product Grid**: Display best-selling products in an organized grid layout
- **Product Details**: Tap on any product to view detailed information
- **Search**: Search bar for finding specific products
- **Bottom Navigation**: Easy navigation with Home, Shopping, and Profile sections
- **Beautiful UI**: Material Design with clean and intuitive interface

## Project Structure

```
lib/
├── main.dart           # App entry point
├── homepage.dart       # Home screen with categories and products
└── detalis.dart        # Product details screen
```

## Getting Started

### Prerequisites

- Flutter SDK (3.10.7 or higher)
- Dart SDK

### Installation

1. Clone the repository:

```bash
git clone https://github.com/MostafaAbdallah2/Categories.git
cd Categories
```

2. Get dependencies:

```bash
flutter pub get
```

3. Run the app:

```bash
flutter run
```

## Dependencies

- **flutter**: Flutter SDK
- **cupertino_icons**: iOS-style icons
- **flutter_lints**: Code style linting

## Build

### Android

```bash
flutter build apk
```

### iOS

```bash
flutter build ios
```
